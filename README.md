## 🧠 What You'll Learn

### 🔤 Variables – `var`, `let`, `const`
- Differences in scope, hoisting, and reassignment
- When to use each declaration type

### ➕ Operators & Type Coercion
- Arithmetic operators: `+`, `-`, `*`, `/`, `%`, `**`
- Why `"10" + 3 = '103'` but `"10" - 3 = 7`
- Pre/Post Increment & Decrement (`++x` vs `x++`)
- `Math.pow()` vs `**`

### 🔡 String Functions
- Common built-in string methods
- Manipulation, searching, and formatting strings

### 🧩 Objects & Functions
- Creating and accessing objects
- Methods, `this` keyword, and object patterns

### 🗺️ Map & Set
- `Map` — key-value pairs with any key type
- `Set` — unique value collections
- Methods: `.set()`, `.get()`, `.has()`, `.delete()`, `.clear()`, `.size`

### 📅 Date & Date Time
- Creating dates with `new Date()`
- Getter/Setter methods: `getDate()`, `getMonth()`, `setFullYear()`, etc.
- Formatting with `toLocaleDateString()` and locale tags

### ⚖️ Equality Operators
- `==` (loose) vs `===` (strict) vs `Object.is()`
- Edge cases: `NaN === NaN` → `false`, `+0 === -0` → `true`
- When and why to use `Object.is()` and `Number.isNaN()`
